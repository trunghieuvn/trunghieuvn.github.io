Feature demo

One link, one QR code. When a visitor opens gobrain.fighttech.vn/get/ (or scans the QR), the page auto-detects their platform and forwards them to the right store — App Store on iOS, Google Play on Android. Desktop visitors get a chooser.

How it works

// Runs in <head>, before paint, so phones never see a flash.
var ua = navigator.userAgent;
var isAndroid = /Android/i.test(ua);
var isIOS = /iPad|iPhone|iPod/i.test(ua) ||
  (navigator.platform === "MacIntel" && navigator.maxTouchPoints > 1);

if (isAndroid) location.replace(PLAY_STORE_URL);
else if (isIOS) location.replace(APP_STORE_URL);
// else: desktop — show the store chooser instead.
